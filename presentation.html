02 THE PROBLEM

Municipal complaint systems are broken

  • 🕳️
    No Transparency — Citizens file complaints and never hear back. Complaints disappear into a black hole.
  • 🐌
    Slow Routing — Nobody knows which department should handle the issue. Complaints sit for weeks.
  • ⚖️
    No Prioritization — A dangerous live wire gets the same urgency as a cracked pavement tile.
  • 📋
    Duplicate Flooding — 20 people report the same pothole, creating 20 separate tickets and wasting resources.
  • 🤥
    No Proof of Repair — Departments mark issues "fixed" without any evidence.
  • 🔒
    No Citizen Recourse — Once "closed", there's no way to reopen even if the repair was bad.
03 OUR SOLUTION

CivicResolve AI fixes all of it

One platform that connects Citizens → AI → Admin → Departments → back to Citizens.

🤖

AI Auto-Classification

Gemini AI reads the complaint and instantly identifies category, severity & the correct department.

📊

Priority Scoring

Formula-based ranking (0–100) so critical hazards like electrical risks always come first.

🔍

Duplicate Detection

Reports within 150 meters of each other for the same category get auto-clustered together.

📸

Photo Verification

Departments must upload repair proof. Admin compares Before vs After photos side-by-side.

🔁

Citizen Reopen

If the fix is bad, citizens can reopen the case with a justification and it goes back to admin.

📝

Full Audit Trail

Every action is permanently logged as an immutable event. Nothing gets deleted or tampered.

10 AI ENGINE

🧠 Gemini AI Under the Hood

What AI Does

  • ✓ Reads citizen description & analyzes photo
  • ✓ Predicts category (POTHOLE, ELECTRICAL_HAZARD, etc.)
  • ✓ Assigns severity (1–5)
  • ✓ Evaluates safety risk (LOW → CRITICAL)
  • ✓ Suggests department routing
  • ✓ Flags sensitive content
  • ✓ Returns confidence score

Safety Guardrails

  • ⚡ Graceful Fallback — No API key? Rule-based heuristics take over. App never breaks.
  • ⚡ Human-in-the-Loop — AI only recommends. Admin always makes the final decision.
  • ⚡ Clearly Labeled — All AI outputs carry an explicit "AI Recommendation" badge.
  • ⚡ Server-Side Only — AI runs on the backend. No client-side API key exposure.
Priority Score = (Severity × 12) + Category Weight + Evidence Pts + Upvote Bonus
